## About 
A simple starter with netlify-cms, stripped down bootstrap grid and more. All in a nice and clean prefilled structure. More details to-be-written.

## Install

Basic
  1. Download Repo
  2. `yarn install`
  3. Push to new git repo

Netlify
  1. New Site from Git
  2. Enable Identity
  3. Enable Git Gateway
  4. Invite CMS
